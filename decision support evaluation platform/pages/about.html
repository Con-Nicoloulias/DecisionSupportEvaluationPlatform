About Decision Support Evaluation Platform
The aim of this platform is to evaluate different decision support environments.

Instructions
The navigation pane will direct the user to each of the following links:

Game List
All games that are created will be listed on this page, with each game having their own unique 'gameID' and 'status'. Clicking on 'Create New Game' will create a new game that will appear under the following interfaces 'Graphic Interface' or 'Text Interface'. Clicking on a game under the following interfaces will direct the user to play the same game on a unique interface. The 'status' of the game will be changed when a user has completed the game. This will transfer the game from 'Game List' to 'Completed List'.

Completed List
This link displays a list of all the games that are completed under the corresponding interface that was used to play the game. Clicking on a game will direct the user to do a playthrough. The user can click 'Next' to see the next move made, as well as click 'Back' to see a previous move made. The 'Switch UI' button will direct the user to playthrough a completed game on different interfaces.
